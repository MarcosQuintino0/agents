# Skills

As skills sao a camada de comportamento do ecossistema. Elas orientam a IA sobre escopo, seguranca, comandos, artefatos e criterios de pronto.

## Resumo

| Skill | Papel |
| --- | --- |
| [qa-api](qa-api.md) | Trilha oficial de testes Cypress/API. |
| [qa-api-fuzz](qa-api-fuzz.md) | Fuzzing investigativo e exploracao de robustez. |
| [qa-debug-report](qa-debug-report.md) | Debug manual de falhas reais com FailLens. |
| [qa-chamado](qa-chamado.md) | Rascunhos de chamados a partir de problemas numerados. |
| [graphify](graphify.md) | Validacao e execucao da versao travada do Graphify. |

## Padrao de uma skill

Cada pagina de skill documenta:

- objetivo;
- quando usar;
- quando nao usar;
- comandos;
- prompts;
- artefatos;
- relacao com outras skills;
- regras de seguranca.
