# Schemas

JSON Schemas used to validate deterministic artifacts produced by the skill.

- `rule-registry.schema.json`: confirmed API rules and evidence source.
- `mass-map.schema.json`: context, data, setup, cleanup, and restoration map.
- `generated-rules.schema.json`: generated rules artifact.
- `generated-scenarios.schema.json`: scenario matrix artifact.

These schemas do not define backend API contracts. They define the shape of skill planning artifacts.
