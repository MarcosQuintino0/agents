# Templates

Templates auxiliares usados pela skill quando o agente precisa gerar artefatos textuais.

- `issue-bdd-template.md`: base para rascunho de chamado/defeito quando a analise de report tiver evidencia suficiente e o usuario solicitar a criacao.

Nao trate estes templates como contrato de API. Eles sao apenas modelos de escrita.
