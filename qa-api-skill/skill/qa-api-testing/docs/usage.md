# Usage

Install the package in the consumer project:

```bash
npm install -D @empresa/qa-api-skill
```

Install the skill files:

```bash
npx qa-api-skill init
```

Update an existing installed skill with backup:

```bash
npx qa-api-skill update
```

Check environment:

```bash
npx qa-api-skill doctor
```

Use the skill in Codex:

```text
Use a skill qa-api-testing para criar testes de API para a API Empresas.
```
